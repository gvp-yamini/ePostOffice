Following is a list of functionality of the system:

Philately

This screen will show all the types of Stamps with their values and their history. Provide the image of each of the stamp types. On selecting any one of the Stamp type, we can know everything about it. It also gives rates for special covers and all the Philately offices.

Pickup service

Through which one can request to pick up packages of different sizes from their home over the internet. User can only view his/her past requests of this service along with their details.

Address Book

In the case of the transfer of the user to new address, one can register their new address here and the shipment of the postal service will be delivered to their updated address. The user should register along with their address in order to use this service. Registered users can search for others address by specifying any two fields like first name and street name.

Electronic Money Order (eMO)

eMO is a web based rapid money transfer service offered by India Post between two individuals within India.It provides the following to user eMO Booking Procedure,eMO Delivery Procedure,eMO make Payment form,Frequent Questions on eMO.

Information

This Provides various useful information for the users related to the postal services like Pin code search,Speed Post Track and Trace,Banking, Insurance.
